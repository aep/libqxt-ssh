Detailed Description

Execute a Job on a QThread. once or multiple times.

QxtJob allows easily starting jobs on different threads.
exec() will ask for the QThread to run the job on. The Qthread needs an event loop. 4.3 and later versions of Qt have a non virtual QThread with a default event loop, allowing easy deployment of jobs.

QThread thread;
thread.start();
LockJob().exec(&thread);
