Getting Started

At first you need qmake. You get it from the qt toolkit by TrollTech

Get the sources

you can either use a release package from the download section or get it from svn 
        svn co https://libqxt.svn.sourceforge.net/svnroot/libqxt/trunk libqxt

Configuration

in the projects root there is a file called config.pri, open it and you will see the different options you may configure

Installation Path

set QXTINSTALLDIR to where you want to install libqxt. for unix the librarys will also be installed in /usr/lib, the docs in /usr/doc and the includes in /usr/include if you want another prefix then /usr define QXTUNIXPREFIX

Modules

the variable QXT contains all the modules you want similar to the QT variable of qt. eg if you don't want the sql module type QXT-=sql or if you want just the core type QXT=core

Depencys

libqxt depends on several third party libs. if you don't want that you may use the QXT_LIBS variable to tell the buildsystem to exlcude these librarys.
Note:
if you exclude librarys the features depending on the libraray will not be available

Stability

you may configure the desired state of stability by the variable QXT_stability. Developers mark their packages to be stable or broken on specific platforms.
Note:
if you have custom mkspecs this will not work properly. check the next section for a workaround

Ignore Stability

undependend from the above declaration you may set QXT_ignore_stability to compile a package regardless what its stability is set. This couses aditional abi incompatibility and should only used if realy necesary. If a package has not yet been set to stable but you find it works very well, please make a bug report.

Making

make sure your qt environment is propably setup.

type 

qmake && make
and fetch some coffee

Installing

type as root

make install

Note:
this will also install qxt.prf to your qtdir/mkspecs/features if you ever change your qt you need to copy that file

Using Qxt in your project

add 
CONFIG  += qxt
QXT     += core media
to your project file. the QXT variable defines the modules you want. In this case core and media are used
Note:
modules may depend on other modules. these are automaticly added.
