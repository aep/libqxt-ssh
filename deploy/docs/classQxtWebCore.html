Detailed Description

qxtweb application core class. communicates, delegates, does all of the magic ;)

QxtWebCore is the base class of your web application. it listens to the scgi protocoll

construct one webcore object in the main function of your application. you must contruct it AFTER QCoreApplication and BEFORe any controllers.

        int main(int argc,char ** argv)
                {
                QCoreApplication  app(argc,argv);
                QxtWebCore core();
                core.listen(8080);
                QxtWebController controller("root");
                app.exec();
                }
