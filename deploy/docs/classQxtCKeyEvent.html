Detailed Description

Describes a keyboard input event.

The QxtCKeyEvent class describes a keyboard input event.

Constructor & Destructor Documentation

QxtCKeyEvent::QxtCKeyEvent ( int  qtKey, Qt::KeyboardModifiers  mods = Qt::NoModifier, int  rawKey = 0 ) [inline]

Creates a new QxtCKeyEvent.

The qtKey values can be found in the Qt::Key enum. It may be Qt::Key_unknown, meaning that the ncurses key code (passed via rawKey) could not be mapped to a Qt key code.


Member Function Documentation

int QxtCKeyEvent::key ( ) const [inline]

Returns a value from the Qt::Key enum corresponding to the key that triggered the event.


Qt::KeyboardModifiers QxtCKeyEvent::modifiers ( ) const [inline]

Returns the keyboard modifiers pertaining to the keystroke that triggered the event.

Note that not all modifiers or combinations of modifiers are available from the curses backend.


int QxtCKeyEvent::nativeScanCode ( ) const [inline]

Returns the native curses key code for the key that triggered the event.
