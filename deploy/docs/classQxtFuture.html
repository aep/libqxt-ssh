QVariant QxtFuture::delayedResult ( int  msec = -1  ) 

asks for the result of the execution.
waits until the done() signal occured or return a QVariant() if the timout ocures earlier
This uses QxtSignalWaiter so it will _not_ block your current thread.

Warning:
this function is not reentrant. You have been warned
